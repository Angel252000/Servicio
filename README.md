# SpotLight - Venta de Vehículos Eléctricos

Plataforma profesional para la comercialización de vehículos eléctricos de alta gama. Este proyecto fue desarrollado para la autogestión y muestra de un catálogo especializado.

## Características
- **Catálogo Dinámico:** Visualización detallada de modelos premium.
- **Navegación Intuitiva:** Menú de acceso rápido a secciones clave.
- **Diseño Moderno:** Interfaz enfocada en la experiencia de usuario y lujo.

## Tecnologías
- HTML5 / CSS3
- JavaScript (Modern ES6+)
- Git para control de versiones

## Instrucciones de uso
1. Clona este repositorio.
2. Abre el archivo `about.html` en tu navegador.
3. Explora la navegación y las secciones integradas.
